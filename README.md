# US Equity Discipline

改訂版110点方式（V5）で199銘柄、7指数、15セクターを毎日採点し、規律可能性の推移も表示する静的サイトです。

公開URL: https://raw.githack.com/kazumaobara0726-oss/-ks-view-7f3c91/main/index.html

- 個別株は指定された重複なしの199銘柄を母集団とし、画面には上位20銘柄を表示
- ランキング外も検索欄から表示
- 日足100点×50％＋週足100点×50％＋月足構造ボーナス最大10点
- 日足・週足は直線上昇・高値圏上昇35、出来高30、下方向25、ATH位置・移動平均線構造10の4領域
- 直線上昇35点の中で、直近20期間の急騰後を−5〜＋4点補正し、最終点を0〜35点に制限
- 反発は5つの確認条件のうち2つ以上かつ2期間以上で判定し、単独の大陰線による二重減点を抑制
- 過去スコアは各時点までに判明していたデータだけで再計算し、未来の価格・出来高を参照しない
- 詳細画面は6か月を標準に1か月・3か月・6か月・1年・3年を切り替え、縦軸0〜110固定で表示
- 実線は最終スコア、破線は減点・上限適用前。点数帯背景、状態変化マーカー、5日平均の任意表示に対応
- 一覧画面には各銘柄の直近60営業日ミニ折れ線を表示
- 細かい配点・補正・上限監査は初期状態で閉じ、必要なときだけ展開
- 白背景のレスポンシブ表示でPC・スマートフォンの両方に対応
- 短期ランダム、規律崩れ、バブル的急騰、長期下降を別々に判定
- 画面のチャートは日足・週足のみ
- GitHub Actionsが毎日23:35 UTCに最新完成足を収集
- `robots.txt` と各ページの `noindex` で検索エンジンへの掲載を抑止

## ローカル更新

```bash
python scripts/update_data.py
python -m http.server 8000
```

データはYahoo Financeの公開チャートエンドポイントから取得します。価格はリアルタイム値ではなく、各市場の最新完成日足終値です。

このサイトは計算式による規律性の分析・学習補助用であり、売買の指示や推奨を行うものではありません。
