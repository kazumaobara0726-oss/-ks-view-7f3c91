# 規律可能性スコア

確定版110点方式で200銘柄、7指数、15セクターを毎日採点する静的サイトです。

公開URL: https://raw.githack.com/kazumaobara0726-oss/-ks-view-7f3c91/main/index.html

- 個別株は指定された重複なしの200銘柄を母集団とし、画面には上位20銘柄を表示
- ランキング外も検索欄から表示
- 日足100点×50％＋週足100点×50％＋月足構造ボーナス最大10点
- 単発急落、短期ランダム、規律崩れ、バブル的急騰、長期下降を別々に判定
- 画面のチャートは日足・週足のみ
- GitHub Actionsが毎日23:35 UTCに最新完成足を収集
- `robots.txt` と各ページの `noindex` で検索エンジンへの掲載を抑止

## ローカル更新

```bash
python scripts/update_data.py
python -m http.server 8000
```

データはYahoo Financeの公開チャートエンドポイントから取得します。価格はリアルタイム値ではなく、各市場の最新完成日足終値です。
